# Тернарный оператор

Сокращениие как альтернатива if/else

Тернарный оператор `? :` — это компактная замена условию `if-else` **для выражений**, где нужно выбрать одно из двух значений. 

**Синтаксис:**
```cpp
условие ? значение_если_истина : значение_если_ложь
```

### Как работает ваш пример:
```cpp
abs(L - round(L)) < 1e-5 ? L : -1
```
1. **Проверка условия:** `abs(L - round(L)) < 1e-5`  
   - Сравнивает разницу между `L` и его округлённой версией с числом `0.00001`.
   - Если разница меньше — число `L` считается "целым" (дробная часть ничтожна).

2. **Результат:**
   - **Если условие истинно** → возвращает `L`.
   - **Если ложно** → возвращает `-1`.

### Другие примеры:
1. **Простой выбор:**
   ```cpp
   int x = (a > b) ? a : b; // Выбирает максимум из a и b
   ```

2. **Проверка чётности:**
   ```cpp
   string result = (num % 2 == 0) ? "чётное" : "нечётное";
   ```

3. **Работа со строками:**
   ```cpp
   string name = (isUser) ? user_name : "Гость";
   ```

### Особенности:
- **Всегда возвращает значение**, поэтому можно использовать в выражениях:
  ```cpp
  cout << (x > 0 ? "положительное" : "отрицательное");
  ```
- **Типы значений должны быть совместимы** (или приводиться к общему типу).
- **Условие вычисляется только один раз**.

### Важно:
- Используйте для простых проверок. Если логика сложная — лучше подойдёт `if-else`.
- Не злоупотребляйте вложенными тернарными операторами — это ухудшает читаемость.
