# Поиск в отсортированной по строкам и столбцам матрице

Дана матрица, в которой каждая строка и каждый столбец отсортированы по возрастанию. Нужно эффективно найти элемент.

**Оптимальный подход (O(rows+cols)):** Начать поиск с правого верхнего (или левого нижнего) угла.

* Если текущий элемент равен искомому — мы его нашли.
* Если текущий элемент больше искомого — значит, в этом столбце его быть не может (все элементы ниже будут ещё больше), двигаемся **влево**.
* Если текущий элемент меньше искомого — значит, в этой строке его быть не может (все элементы левее будут ещё меньше), двигаемся **вниз**.

```c++
#include <iostream>
#include <vector>

int main() {
    std::vector<std::vector<int>> matrix = {
        {10, 20, 30, 40},
        {15, 25, 35, 45},
        {27, 29, 37, 48},
        {32, 33, 39, 50}
    };
    // Эта матрица отсортирована и по строкам, и по столбцам

    int value_to_find;
    std::cout << "Введите значение для поиска в матрице: ";
    std::cin >> value_to_find;

    int rows = matrix.size();
    int cols = matrix[0].size();

    int i = 0;       // Начинаем с первой строки
    int j = cols - 1; // ... и последнего столбца

    bool found = false;
    while (i < rows && j >= 0) {
        if (matrix[i][j] == value_to_find) {
            std::cout << "Элемент найден в позиции (" << i << ", " << j << ")" << std::endl;
            found = true;
            break;
        } else if (matrix[i][j] > value_to_find) {
            j--; // Двигаемся влево
        } else {
            i++; // Двигаемся вниз
        }
    }

    if (!found) {
        std::cout << "Элемент не найден." << std::endl;
    }

    return 0;
}
```
