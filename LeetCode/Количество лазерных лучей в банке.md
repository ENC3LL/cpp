# Количество лазерных лучей в банке

В банке активирована охранная система. Вам дан двумерный двоичный массив `bank` размером `m x n`, который представляет собой поэтажный план банка. `bank[i]` представляет собой `i^{ую}` строку, содержащую `'0'` и `'1'`. `'0'` означает, что ячейка пуста, а `'1'` означает, что в ячейке есть охранное устройство.

Между **двумя** охранными устройствами есть **один** лазерный луч, если выполняются **оба условия**:

* Два устройства расположены в двух **разных строках**: `r1` и `r2`, где `r1 < r2`.
* Для каждой строки `i`, где `r1 < i < r2`, нет **охранных устройств** в `i^{ой}` строке.

Лазерные лучи независимы, то есть один луч никак не взаимодействует с другим.

Верните *общее количество лазерных лучей в банке*.


**Пример 1:**

![image](./.assets/image-1761589548805.png)

**Ввод:** bank = ["011001","000000","010100","001000"]
**Вывод:** 8
**Пояснение:** Между каждой из следующих пар устройств имеется один луч. Всего лучей 8:
 * bank[0][1] -- bank[2][1]
 * bank[0][1] -- bank[2][3]
 * bank[0][2] -- bank[2][1]
 * bank[0][2] -- bank[2][3]
 * bank[0][5] -- bank[2][1]
 * bank[0][5] -- bank[2][3]
 * bank[2][1] -- bank[3][2]
 * bank[2][3] -- bank[3][2]
Учтите, что между устройствами в
```
0^{й}
```

 и в 
 ```
3^{й}
```

 строках лучей нет.
Это связано с тем, что
```
2^{я}
```

 строка содержит охранные устройства, что противоречит второму условию.


**Пример 2:**


![image](./.assets/image-1761589599097.png)



**Ввод:** bank = ["000","111","000"]
**Вывод:** 0
**Пояснение:** Не существует двух устройств, расположенных в двух разных строках.

# Решение
```c++
class Solution {
public:
    int numberOfBeams(vector<string>& bank) {
        int ans = 0, prev = 0;
        for (auto& row : bank) {
            int device = count(row.begin(), row.end(), '1');
            if (device > 0) {
                ans += exchange(prev, device) * device;
            }
        }
        return ans;
    }
};
```
