# Переверните целое число
с проверкой на 32-бита

Дано 32-битное целое число `x` со знаком, верните *перевернутое число* `x`. Если переворачивание `x` приводит к выходу значения за пределы 32-битного целого числа со знаком (`[-2^31, 2^31-1]`), то верните 0.

**Предположим, что среда выполнения кода не позволяет вам хранить 64-битные целые числа (знаковые или беззнаковые).**

**Пример 1:**

**Ввод:** x = 123
**Вывод:** 321**Пример 2:**

**Ввод:** x = -123
**Вывод:** -321**Пример 3:**

**Ввод:** x = 120
**Вывод:** 21

# Решение

```c++
class Solution {
public:
    int reverse(int x) {

        string str = to_string(x);
        if (str[0] == '-') {
            ::reverse(str.begin(), str.end());
            str = '-' + str;
        }
        else {
            ::reverse(str.begin(), str.end());
        }

        try {
            int m = stoi(str);
            return m;
        }
        catch (const std::out_of_range& e) {
            return 0;
        }
        catch (const std::invalid_argument& e) {
            return 0;
        }
    }
};
```
