# Простая банковская система

Перед вами поставлена задача написать программу для популярного банка, которая автоматизирует все входящие операции (перевод, депозит и вывод средств). В банке есть счета, пронумерованные от до . Начальный баланс каждого счета хранится в **индексированном 0** целочисленном массиве , при этом начальный баланс счета равен .`n``1``n``balance``(i + 1)th``balance[i]`

Выполните все **действительные** транзакции. Транзакция **действительна**, если:

* Заданный номер (номера) счета находится между и , и`1``n`
* Сумма снятых или переведенных денег **меньше или равна** остатку на счете.

Реализуем класс:`Bank`

* `Bank(long[] balance)` Инициализирует объект целочисленным массивом **с индексацией 0.**`balance`
* `boolean transfer(int account1, int account2, long money)` Переводит доллары со счета с номером на счет с номером . Возвращается, если транзакция прошла успешно, в противном случае.`money``account1``account2``true``false`
* `boolean deposit(int account, long money)` Внесите доллары на счет с номером . Возвращается, если транзакция прошла успешно, в противном случае.`money``account``true``false`
* `boolean withdraw(int account, long money)` Вывод долларов со счета с номером . Возвращается, если транзакция прошла успешно, в противном случае.`money``account``true``false`

**Пример 1:**

**Input**

# Решение

```c++
class Bank {
private:
    vector<long long> accounts;

public:
    Bank(vector<long long>& balance) : accounts(balance) {}

    bool transfer(int account1, int account2, long long money) {
        int idx1 = account1 - 1;
        int idx2 = account2 - 1;

        if (idx1 < 0 || idx1 >= accounts.size() ||
            idx2 < 0 || idx2 >= accounts.size()) {
            return false;
        }

        if (accounts[idx1] < money) {
            return false;
        }

        accounts[idx1] -= money;
        accounts[idx2] += money;
        return true;
    }

    bool withdraw(int account, long long money) {
        int idx = account - 1;

        if (idx < 0 || idx >= accounts.size() || accounts[idx] < money) {
            return false;
        }

        accounts[idx] -= money;
        return true;
    }

    bool deposit(int account, long long money) {
        int idx = account - 1;

        if (idx < 0 || idx >= accounts.size()) {
            return false;
        }

        accounts[idx] += money;
        return true;
    }
};
```
