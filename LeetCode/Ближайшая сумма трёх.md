# Ближайшая сумма трёх

Дан целочисленный массив длины и целое число , найти три целых числа в таком, чтобы сумма была наиболее близка к .`nums``n``target``nums``target`

Возвращает *сумму трех целых чисел*.

Можно предположить, что каждый вход будет иметь ровно одно решение.

**Пример 1:**

**Input:** nums = [-1,2,1,-4], target = 1
**Output:** 2
**Explanation:** The sum that is closest to the target is 2. (-1 + 2 + 1 = 2).
**Пример 2:**

**Input:** nums = [0,0,0], target = 1
**Output:** 0
**Explanation:** The sum that is closest to the target is 0. (0 + 0 + 0 = 0).

# Решение

```c++
class Solution {
public:
    int threeSumClosest(vector<int>& nums, int target) {
        sort(nums.begin(), nums.end());

        int closestSum = nums[0] + nums[1] + nums[2];

        for (int i = 0; i < nums.size() - 2; i++) {

            int left = i + 1;
            int right = nums.size() - 1;

            while (left < right) {

                int currentSum = nums[i] + nums[left] + nums[right];

                if (abs(currentSum - target) < abs(closestSum - target)) {
                    closestSum = currentSum;
                }

                if (currentSum < target) {

                    left++;
                }
                else if (currentSum > target) {

                    right--;
                }
                else {

                    return currentSum;
                }

            }

        }

        return closestSum;
    }
};
```
