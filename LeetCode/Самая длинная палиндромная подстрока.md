# Самая длинная палиндромная подстрока

Дана строка `s`, верните *самую длинную палиндромную подстроку* в строке `s`.

**Пример 1:**

**Ввод:** s = "babad"
**Вывод:** "bab"
**Пояснение:** "aba" тоже является правильным ответом.**Пример 2:**

**Ввод:** s = "cbbd"
**Вывод:** "bb"

# Решение

```c++
string longestPalindrome(string s) {
    if (s.empty()) return "";
    
    int start = 0, maxLen = 1;
    
    for (int i = 0; i < s.length(); i++) {
        // Для палиндромов нечетной длины
        int len1 = expandAroundCenter(s, i, i);
        // Для палиндромов четной длины
        int len2 = expandAroundCenter(s, i, i + 1);
        
        int len = max(len1, len2);
        if (len > maxLen) {
            maxLen = len;
            start = i - (len - 1) / 2;
        }
    }
    
    return s.substr(start, maxLen);
}

int expandAroundCenter(const string& s, int left, int right) {
    while (left >= 0 && right < s.length() && s[left] == s[right]) {
        left--;
        right++;
    }
    return right - left - 1;
}
```
