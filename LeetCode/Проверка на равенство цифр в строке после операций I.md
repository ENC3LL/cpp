# Проверка на равенство цифр в строке после операций I

Вам дается строка, состоящая из цифр. Повторяйте следующую операцию до тех пор, пока в строке не будет **ровно** две цифры:`s`

* Для каждой пары последовательных цифр в , начиная с первой цифры, вычислите новую цифру как сумму двух цифр **по модулю** 10.`s`
* Замените на последовательность вновь вычисленных цифр, *сохраняя порядок*, в котором они вычисляются.`s`

Возвращается, если последние две цифры в поле **совпадают**; в противном случае возвращаем .`true``s``false`

**Пример 1:**

**Входные данные:** s = "3902"

**Вывод:** true

**Объяснение:**

* Первоначально `s = "3902"`
* Первая операция:* `(s[0] + s[1]) % 10 = (3 + 9) % 10 = 2`
* `(s[1] + s[2]) % 10 = (9 + 0) % 10 = 9`
* `(s[2] + s[3]) % 10 = (0 + 2) % 10 = 2`
* `s` Становится `"292"`
* `(s[0] + s[1]) % 10 = (3 + 9) % 10 = 2`
* `(s[1] + s[2]) % 10 = (9 + 0) % 10 = 9`
* `(s[2] + s[3]) % 10 = (0 + 2) % 10 = 2`
* `s` Становится `"292"`
* Вторая операция:* `(s[0] + s[1]) % 10 = (2 + 9) % 10 = 1`
* `(s[1] + s[2]) % 10 = (9 + 2) % 10 = 1`
* `s` Становится `"11"`
* `(s[0] + s[1]) % 10 = (2 + 9) % 10 = 1`
* `(s[1] + s[2]) % 10 = (9 + 2) % 10 = 1`
* `s` Становится `"11"`
* Поскольку цифры in совпадают, на выходе получается .`"11"``true`

**Пример 2:**

**Входные данные:** s = "34789"

**Вывод:** false

**Объяснение:**

* Первоначально.`s = "34789"`
* После первой операции домен .`s = "7157"`
* После второй операции .`s = "862"`
* После третьей операции .`s = "48"`
* Поскольку , на выходе получается .`'4' != '8'``false`

# Решение

```c++
class Solution {
public:
    bool hasSameDigits(string& s) {
        for (int i = s.size() - 1; i >= 2; i--) {
            for (int j = 0; j < i; j++) {
                s[j] = (s[j] + s[j + 1]) % 10 + '0';
            }
        }
        return s[0] == s[1];
    }
};
```
