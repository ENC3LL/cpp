# Зигзагообразное преобразование



Строка `s = "PAYPALISHIRING"` записывается зигзагом в заданном количестве строк `numRows`, как показано ниже: (вы можете отобразить этот шаблон моноширинным шрифтом для лучшей читаемости)

P      A      H        N
A  P  L  S   I    I    G
Y       I      R    

А потом читается построчно: `"PAHNAPLSIIGYIR"`

Напишите код, который принимает строку `s` и выполняет зигзагообразное преобразование с учетом количества строк `numRows`:

string convert(string s, int numRows); 

**Пример 1:**

**Ввод:** s = "PAYPALISHIRING", numRows = 3
**Вывод:** "PAHNAPLSIIGYIR"**Пример 2:**

**Ввод:** s = "PAYPALISHIRING", numRows = 4
**Вывод:** "PINALSIGYAHRPI"
**Пояснение:** 
P           I           N
A      L   S      I   G
Y  A       H  R
P           I

**Пример 3:**

**Ввод:** s = "A", numRows = 1
**Вывод:** "A"

# Решение 

```c++
string convert(string s, int numRows) {
    if (numRows == 1) return s;
    
    string val;
    int n = s.size();
    int cycle = 2 * numRows - 2; // полный цикл зигзага
    
    // идём "волнами" сверху вниз и обратно
    for (int i = 0; i < numRows; i++) {
        for (int j = 0; j + i < n; j += cycle) {
            // Первый элемент в "столбце" (вертикальное движение)
            val.push_back(s[j + i]);
            
            // Второй элемент (диагональное движение) - только для не-первой и не-последней строки
            if (i != 0 && i != numRows - 1 && j + cycle - i < n) {
                val.push_back(s[j + cycle - i]);
            }
        }
    }
    
    return val;
}
```
