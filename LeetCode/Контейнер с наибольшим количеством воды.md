# Контейнер с наибольшим количеством воды

Вам дан целочисленный массив `height` длины `n`. Нарисовано `n` вертикальных линий, где высота каждой `iой` линии задается двумя конечными точками `(i, 0)` и `(i, height[i])`.

Найдите две линии, которые вместе с осью X образуют контейнер, в котором содержится больше всего воды.

Верните *максимальное количество воды, которое может хранить контейнер*.

**Учтите**, что вы не можете наклонять контейнер.

 

**Пример 1:**

![image](./.assets/image-1761259332438.png)

**Ввод:** height = [1,8,6,2,5,4,8,3,7]
**Вывод:** 49
**Пояснение:** Вышеуказанные вертикальные линии представлены массивом [1,8,6,2,5,4,8,3,7]. В таком случае, максимальное количество воды (синяя область), которое может содержать контейнер, равно 49.**Пример 2:**

**Пример 2:**
**Ввод:** height = [1,1]
**Вывод:** 1

# Решение

```c++
int maxArea(vector<int>& height) {
    int left = 0;
    int right = height.size() - 1;
    int max_area = 0;
    
    while (left < right) {
        // Высота = минимум из двух столбцов, ширина = расстояние между индексами
        int current_area = min(height[left], height[right]) * (right - left);
        max_area = max(max_area, current_area);
        
        // Двигаем указатель с меньшей высотой
        if (height[left] < height[right]) {
            left++;
        } else {
            right--;
        }
    }
    
    return max_area;  // Для [1,8,6,2,5,4,8,3,7] вернет 49
}
```
