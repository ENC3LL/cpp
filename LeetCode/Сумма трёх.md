# Сумма трёх

**В задании допущена логическа ошибка, так что можно забить на i !=k и так далее**

Дан целочисленный массив `nums`, верните все тройки `[nums[i], nums[j], nums[k]]` такие, что `i != j`, `i != k` и `j != k`, а также `nums[i] + nums[j] + nums[k] == 0`.

Учтите, что результирующий массив не должен содержать повторяющихся троек.

**Пример 1:**

**Ввод:** nums = [-1,0,1,2,-1,-4]
**Вывод:** [[-1,-1,2],[-1,0,1]]
**Пояснение:** 
nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0.
nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0.
nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0.
Уникальные тройки - это [-1,0,1] и [-1,-1,2].
Порядок троек и порядок внутри троек не имеют значения.**Пример 2:**

**Ввод:** nums = [0,1,1]
**Вывод:** []
**Пояснение:** Единственно возможная тройка не дает в сумме 0.**Пример 3:**

**Ввод:** nums = [0,0,0]
**Вывод:** [[0,0,0]]
**Пояснение:** Единственно возможная тройка дает в сумме 0.

# Решение

```c++
class Solution {
public:
    vector<vector<int>> threeSum(vector<int>& nums) {
        vector<vector<int>> result;
        sort(nums.begin(), nums.end());

        for (int i = 0; i < nums.size() - 2; i++) {

            if (i > 0 && nums[i] == nums[i - 1]) continue;

            int left = i + 1;
            int right = nums.size() - 1;

            while (left < right) {
                int sum = nums[i] + nums[left] + nums[right];

                if (sum == 0) {
                    result.push_back({ nums[i], nums[left], nums[right] });

                    while (left < right && nums[left] == nums[left + 1]) left++;
                    while (left < right && nums[right] == nums[right - 1]) right--;

                    left++;
                    right--;
                }
                else if (sum < 0) {
                    left++;
                }
                else {
                    right--;
                }
            }
        }

        return result;
    }
};
```
